# OfficeAlarm
=======


## Install
brew install ruby
brew install node
brew install
npm config set registry https://registry.npm.taobao.org
npm install -g cnpm --registry=https://registry.npm.taobao.org


cnpm install electron --save-dev
npm start


## 打包
cnpm install electron-packager --save-dev
